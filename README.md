# 🫙 مدير مبيعات الطحينة

نظام إدارة مبيعات الطحينة والزيت الحار — HTML/CSS/JavaScript خالص، بدون أي framework.

## المميزات
- 📊 لوحة تحكم مع رسوم بيانية
- 📦 إدارة المنتجات والمخزون
- 👥 إدارة العملاء وكشف الحساب
- 🧾 إنشاء الفواتير وطباعتها
- 💰 تحصيل الدفعات وتتبع الديون
- 📈 تقارير مبيعات شاملة
- 💾 البيانات محفوظة في المتصفح (localStorage)
- 🌐 واجهة عربية RTL كاملة
- 📱 متجاوب مع الجوال

## التشغيل
افتح ملف `index.html` في المتصفح مباشرة — لا يحتاج سيرفر!
